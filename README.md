# SmartHome Onboarding MVP

Proyecto React + Vite + TypeScript + Tailwind para mostrar un onboarding responsive en localhost.

## Requisitos
- Node.js 18+ (ideal 20+)
- npm

## Instalar
```bash
npm install
```

## Levantar en local
```bash
npm run dev
```

Luego abrí la URL que te muestre Vite, normalmente:
```bash
http://localhost:5173
```

## Build de producción
```bash
npm run build
npm run preview
```
